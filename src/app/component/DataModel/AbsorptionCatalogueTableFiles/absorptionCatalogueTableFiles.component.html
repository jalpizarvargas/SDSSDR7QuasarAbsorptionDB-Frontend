Data Model for the Catalogue Table Files

Files with namse like "cat_plate_fiber_mjd.dat" are catalogue files.

(Columns in these files are read from left to right)


Rows that Start with Field Number = 1

Summary: QSO-level information.

Column Data
1 Field Number
2 SDSS Plate
3 SDSS Fiber
4 SDSS MJD
5 Quasar Redshift from Schneider et al. 2010
6 Quasar Redshift from Hewett & Wild 2010
7 SDSS i band PSF magnitude

Rows that Start with Field Number = 2

Summary: QSO coordinates and dates of the last absorption extraction run.

Column Data
1 Field Number
2 SDSS ID prefix
3 RA [sexigesimal]
4 Dec [sexigesimal]
5 RA [degrees]
6 Dec [degrees]
7 Pipeline run date
8 Pipeline run time

Rows that Start with Field Number = 3

Summary: system-level information for each group of absorption lines matched to the same redshift.

Column Data
1 Field Number
2 Redshift of absorption line system; A value of ‘-2’ in this column indicates the end of listed detections for the file.
3 Minimum rest frame wavelength observable for this system (Angstroms)
4 Maximum rest frame wavelength observable for this system (Angstroms)
5 Confidence grade, indicating the number of strong, unblended lines from commonly observed ions, detected at the same redshift (‘A’=4+; B= 3; ‘C’= 2; ‘C*’= 2, but missing one line in a CIV or MgII doublet; ‘D’ = 1. See publication for full description.)
6 Type of system detection (Some combination of: ‘M’=MgII, ‘C’=CIV, and ‘F’=FeII)
7 Absorption system velocity in the quasar rest frame, as a fraction of the speed of light (v/c)

Rows that Start with Field Number = 4

Summary: the median, observed frame, two-pixel 1-sigma equivalent width error in each of the wavelength regions listed below in the quasar rest-frame. Entries of -1.0 indicate that the wavelength region is not within the bounds of the spectrum.

Column Data
1 Field Number
2 Median 2-pixel flux uncertainty in the quasar rest frame wavelength range: 1215.7-1240.8 Angstroms
3 Median 2-pixel flux uncertainty in the quasar rest frame wavelength range: 1240.8-1399.8 Angstroms
4 Median 2-pixel flux uncertainty in the quasar rest frame wavelength range: 1399.8-1549.5 Angstroms
5 Median 2-pixel flux uncertainty in the quasar rest frame wavelength range: 1549.5-1908.7 Angstroms
6 Median 2-pixel flux uncertainty in the quasar rest frame wavelength range: 1908.7-2799.1 Angstroms
7 Median 2-pixel flux uncertainty in the quasar rest frame wavelength range: 2799.1-3969.0 Angstroms
8 Median 2-pixel flux uncertainty in the quasar rest frame wavelength range: 3969.0-8200.0 Angstroms

Rows that Start with Field Number = 5

Summary: information for each individual absorption line matched to the same systemic redshift.

Column Data
1 Field Number
2 Observed wavelength (Angstroms)
3 Equivalent width (Angstroms), observer frame
4 Equivalent width uncertainty (Angstroms), observer frame
5 Vacuum rest-frame wavelength (Angstroms)
6 Element name
7 Ion number
8 Lyman-alpha flag (1 = detected within the LyA forest; 0 = detected outside the LyA forest)
9 Redshift offset of the line from the system average redshift (given in Field 3)
10 Velocity offset of the line with respect to the system average redshift (given in Field 3)
11 Signal to noise of the line (equivalent width / uncertainty)
12 Full width at half maximum (pixels)
13 Grade flag (‘*’ = line was used to determine the confidence grade, presented in field 3; ‘0’ = line was not used)
14 (if multiple line IDs are possible) Alternate absorption system redshift
15 (if multiple line IDs are possible) Alternate element name
16 (if multiple line IDs are possible) Alternate ion number
17 (if multiple line IDs are possible) Alternate vacuum rest-frame wavelength (Angstroms)

Rows that Start with Field Number = 8

Summary: information for strong (SNR>5) detected lines that could not be confidently identified.

Column Data
1 Field Number
2 Observed wavelength (Angstroms)
3 Equivalent width (Angstroms), observer frame
4 Equivalent width uncertainty (Angstroms), observer frame
5 Signal to noise of the line (equivalent width / uncertainty)
6 Full width at half maximum (pixels)
