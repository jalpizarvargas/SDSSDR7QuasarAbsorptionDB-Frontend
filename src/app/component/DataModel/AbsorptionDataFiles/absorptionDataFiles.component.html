Data Models for the Absorption Data Files

(Columns in these files are read from left to right)


spec_plate_fiber_mjd_.dat Files

Summary: Calibrated SDSS Quasar Spectrum.


Column Data
1 wavelength [Angstroms]
2 flux [10^-17 erg s^-1 cm^-2 A^-1]
3 flux uncertainty
4 model continuum flux [10^-17 erg s^-1 cm^-2 A^-1]
5 pixel width * (flux uncertainty) / flux

nspec_plate_fiber_mjd_.dat Files

Summary: Normalized Quasar Spectrum


Column Data
1 wavelength [Angstroms]
2 normalized flux
3 normalized flux uncertainty
4 model continuum flux [10^-17 erg s^-1 cm^-2 A^-1]

ewspec_plate_fiber_mjd_.dat Files

Summary: A measurement of the absorption equivalent width at each pixel in the spectrum.


Column Data
1 wavelength [Angstroms]
2 equivalent width [Angstroms]
3 equivalent width uncertainty [Angstroms]

lines_plate_fiber_mjd_.dat Files

Summary: A listing of all detected lines with a signal-to-noise ratio >=3.


Column Data
1 wavelength [Angstroms]
2 equivalent width / equivalent width uncertainty

zidinput_plate_fiber_mjd_.dat Files

Summary: A listing of measurements for all detected lines with a signal-to-noise ratio >=3.


Column Data
1 wavelength [Angstroms]
2 pixel width [Angstroms]
3 equivalent width [Angstroms]
4 equivalent width uncertainty [Angstroms]
5 full width half maximum [Angstroms]
