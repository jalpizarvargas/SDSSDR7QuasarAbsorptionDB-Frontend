Data Model for the QSOALS FITS Catalogue


FITS Header Description
plate SDSS Spectroscopic Plate Number
fiber SDSS Spectroscopic Fiber Number
mjd Modified Julian Date
z_qso QSO redshift (Hewett & Wild 2010)
imag QSO i-band apparent magnitude
BAL_flag Broad Absorption Line (BAL) QSO flag:

0 = not identified as a BALQSO
1 = BAL identified (Gibson et al. 2009)
10 = BAL identified (D. VandenBerk, private comm.)
20 = Ambiguous BAL candidate (D. VandenBerk)
BAL_flag2 Alternate BALQSO flag:

0 = not identified as a BALQSO
1 = BAL identified by Gibson et al. (2009)
2 = BAL identified by Shen et al. (2011) only
3 = BAL identified by S. Wesolowski only
4 = BAL identified by both Shen and Wesolowski
target_FIRST FIRST target flag:

1 = targeted
0 = not targeted
20cm_flux FIRST 20cm Flux
20cm_snr FIRST 20cm SNR
sep_FIRST FIRST Target Separation (")
iab_Mag QSO i-band absolute magnitude
z_fgal Redshift of foreground galaxy detected in emission.
spec_aveSNR Mean SNR of Full Spectrum
spec_medSNR Median SNR of Full Spectrum
spec_aveSNR_red Mean SNR > 7200 Å
spec_medSNR_red Median SNR > 7200 Å
ra_hex RA (Hexigesimal)
dec_hex Dec (Hexigesimal)
ra_deg RA (Degrees)
dec_deg Dec (Degrees)
run_date Absorption Pipeline Run Date
w_limits Absorption equivalent width detection limits:

Median two pixel 1σ error in each of the wavelength regions listed below in the QSO rest-frame.
Entries of -1.0 indicate that the wavelength region is not within the bounds of the spectrum.
[1215.7 - 1240.8 Å]
[1240.8 - 1399.8 Å]
[1399.8 - 1549.5 Å]
[1549.5 - 1908.7 Å]
[1908.7 - 2799.1 Å]
[2799.1 - 3969.0 Å]
[3969.0 - 8200.0 Å]
[8200.0 - 9200.0 Å]
num_5sigma_unided_inLyA Number of absorption lines detected at > 5σ within the Ly-α forest
num_5sigma_unided_notinLyA Number of absorption lines detected at > 5σ outside the Ly-α forest
z_abs Absorption system redshift
grade Letter quality grade:

A = 4 lines, W/σW ⩾ 4

B = 3 lines, W/σW ⩾ 4

C = 2 lines, W/σW ⩾ 4
(C IV or Mg II doublet)

C* = 2 lines, W/σW ⩾ 4
(1 line of CIV or Mg II doublet and 1 other ion)

C** = 2 lines, W/σW ⩾ 4
(neither line identified as C IV or Mg II)

D = 1 line, W/σW ⩾ 4; and 1 line, W/σW ⩾ 3 and W/σW < 4

D* = 2 lines, W/σW ⩾ 3 and W/σW < 4

E = 1 line, W/σW ⩾ 4

E* = 1 line, W/σW ⩾ 3 and W/σW < 4
type Code describing the method(s) of detection:

M = system detected by search for Mg II
C = system detected by search for C IV
F = system detected by search for Fe II
sys_num Running count of systems detected in the spectrum, in order of detection
beta v/c of absorption system in the QSO rest frame
DLA_flag_ND2012 Flag indicating the presence of a DLA at z_abs, as determined by Noterdaeme et al. (2012b)
DLA_logNHI_ND2012 N[HI] of DLA from Noterdaeme et al. 2012
DLA_logNHI_err_ND2012 N[HI] error of DLA from Noterdaeme et al. 2012
DLA_zabs_ND2012 Redshift of DLA from Noterdaeme et al. 2012
MgII_class_score Classification score for Mg II detected systems
CIV_class_score Classification score for C IV detected systems
other_class_score Classification score for Fe II-detected systems
lambda_vac Array of laboratory wavelengths (Å) for documented absorption lines, truncated to one decimal:

[1025.7, 1215.7, 1238.8, 1260.4, 1302.2, 1304.4, 1334.5, 1393.8, 1402.8, 1526.7, 1548.2, 1550.8,
1608.5, 1656.9, 1670.8, 1854.7, 1862.8, 2026.1, 2056.3, 2260.8, 2374.5, 2382.8, 2586.7, 2594.5,
2600.2, 2796.4, 2803.5, 2853.0, 3242.9, 3934.8, 3969.6, 5891.5]
lambda_obs Array of observed wavelengths (Å) for each of the ions listed in the lambda_vac array
EW_obs Array of observer-frame equivalent widths (Å) for each of the ions listed in the lambda_vac array
EW_err_obs Array of 1σ errors (Å) on the equivalent widths for each of the ions listed in the lambda_vac array
lya_flag Array of flags indicating detections in the Lyα forest for each of the ions
listed in the lambda_vac array

0 = longward of Lyα (λobs > 1215.7 × zqso)
1 = shortward of Lyα (λobs ⩽ 1215.7 × zqso)

for each of the ions listed in the lambda_vac array
deltav_sys Array of velocity differences (km s−1) between each line and the average redshift of the system
for each of the ions listed in the lambda_vac array
fwhm Array of FWHM measurements (pixels) for each of the ions listed in the lambda_vac array
gradeflag Array of flags denoting lines counting toward the grade for each of the ions
listed in the lambda_vac array

0 = Line is not used for grading
1 = Line is used for grading
blendflag Flag denoting blended lines for each of the ions listed in the lambda_vac array
